## 说明

一款基于Netty 构建的RPC框架，尚处于开发阶段，仅完成部分特性,支持Spring应用，依赖于Spring环境，提供了更简单的服务访问机制

无需配置大量的配置文件和代码，仅需要引入依赖，添加注解，即可实现RPC服务访问


## 依赖库

+ Slf4j
+ Netty4
+ hutool
+ Spring-Context 5.x

## 特性

- [x] 支持 Spring 应用
- [x] 支持多种注册中心
- [x] 快速响应，
- [x] 注解式驱动，无侵入代码设计
